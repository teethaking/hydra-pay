# 🌊 Hydra-Pay: Programmable Waterfall Streams

A modular extension for the **Drips Network** to manage complex capital coordination.

## 🚀 Features
- **Waterfall Logic:** Split incoming streams into multiple vaults.
- **Developer First:** Lightweight SDK for easy integration.
- **Drips v2 Ready:** Built to sit on top of the Drips Hub.

## 🤝 Contributing

- Use **Conventional Commits**.
